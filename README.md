# Explainable Machine Learning for Predicting Childhood Anemia

This repository contains the Python scripts used in the study:

"Explainable Machine Learning for Predicting Childhood Anemia in Sub-Saharan Africa Using DHS Data (2016–2024)"

Data are not included because DHS datasets require authorization.

Researchers can request the datasets from:

https://www.dhsprogram.com/data/

After downloading the DHS datasets, run the scripts in the following order:

model_training.ipynb
